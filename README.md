# 📊 Análise de Dados Agrícolas

Dashboard completo de análise de dados do setor agrícola brasileiro.

## 🚀 Tecnologias
- Python 3
- PostgreSQL
- Power BI

## 📈 O que o projeto faz
- Coleta dados de produção, custos e preços de mercado
- Calcula receita, custo e lucro por produto
- Gera previsão de lucro para 2025 e 2026
- Exporta CSV para dashboard no Power BI

## 💡 Insights encontrados
- Café é o produto mais lucrativo (R$ 19.000)
- Cana-de-açúcar gera prejuízo (-R$ 7.500)
- Lucro total da operação: R$ 100.800

## ▶️ Como executar
```bash
python3 analise.py
```
